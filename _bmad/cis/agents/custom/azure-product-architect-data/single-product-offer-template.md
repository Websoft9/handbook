# Single Product Offer Template

This template is for single-software self-hosted solutions (e.g., "WordPress on Websoft9", "Magento on Websoft9").

## Structure Guidelines

**Emphasis:** Self-hosting benefits, managed service, autonomy, "turnkey solution", data sovereignty

**SKU Strategy:** Different tiers/specs (Starter, Business, Enterprise)

---

## TITLE

**Format:** `[Software] on Websoft9 - Managed Self-Hosting on Azure`

**Example:** `WordPress on Websoft9 - Managed Self-Hosting on Azure`

**Best Practices:**
- Software name first (brand recognition)
- "on Websoft9" emphasizes service provider
- "Managed Self-Hosting" unique positioning
- "on Azure" platform indicator

---

## SUMMARY

**Format:** `Self-host [Software] with full control + expert support. One-click deployment, automated backups, 24/7 managed service. [Primary benefit in numbers].`

**Example:** `Self-host WordPress with full control + expert support. One-click deployment, automated backups, 24/7 managed service. 99.9% uptime guaranteed.`

**Best Practices:**
- Lead with "self-host" + software name
- "Full control + expert support" = unique value
- Quantified benefit or guarantee
- 150-200 characters

---

## SHORT DESCRIPTION

**Format:** `Turnkey [Software] self-hosting on Azure. Pre-configured [Software] with Websoft9 control panel, automated backups, security updates, and 24/7 support. Full control without the hassle.`

**Example:** `Turnkey WordPress self-hosting on Azure. Pre-configured WordPress with Websoft9 control panel, automated backups, security updates, and 24/7 support. Full control without the hassle.`

**Best Practices:**
- "Turnkey" signals ease of use
- List 4-5 key service features
- "Full control without hassle" = positioning
- 200-250 characters
