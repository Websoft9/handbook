# Category Suite Plan Template

For individual app SKUs within a category suite.

**Strategy:** Each Plan = One App. Focus on specific tool value and use cases.

---

## PLAN NAME (3-5 words)

**Format:** `[App Name] [Category] Suite`

---

## PLAN SUMMARY (50-80 chars)

**Format:** `Self-hosted [App] for [primary use case] with one-click deployment and managed support`

---

## PLAN DESCRIPTION

**Note:** Use `html-plan-category-suite-app.html` for actual generation.

### Structure:
1. **Target + Value:** For [persona] building [outcome]
2. **What You Get:** 4-5 bullet points (pre-configured app, Websoft9 panel, backups, updates, support)
3. **Time to Value:** Deploy in [X] minutes
