# 产品



## 使命

Websoft9 产品团队创造客户喜欢且有价值的产品，并在使用产品的生命周期过程提供良好的用户体验：

* 我们的组织目标是创建一个产品经理 （PM） 系统，最大限度地提高新产品、新功能和新团队成员的成功机会。
* 从外部考虑客户问题，而不是从内部考虑技术。
* 用户体验与产品同等重要，不要忘记产品和网站，定价和包装，文档，销售，支持等之间的联系
* 产品为客户创造价值是我们存在的理由

我们的产品是基于在线交易的软件产品，因此其表现形态是多样化的，包括：

- 品牌
- 购买体验感
- 源码
- 镜像
- 文档
- 视频
- 内容营销
- 技术支持
- 服务产品
- 商务
- 群运营（活动、优惠券）
- 活动
- 企业社会责任
- 信息收集：填需求送优惠券
- 其他
