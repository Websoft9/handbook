# 法务

Websoft9 是一家全球化运作的企业，合规性是企业战略和发展之基石。

## 做什么？

企业法务主要涉及如下事务：

* 行政监管法律事务（简称：ToG）：符合政府监管的约束事务，包括网络安全、劳动关系、税务、工商监管等
* 业务经营法律事务（简称：ToB）：业务开展过程中涉及相关的法律事务，有知识产权、消费者权益、产品许可、交易合同等
* 战略管理法律事务（简称：ToS）：对企业发展中重大问题的决策作出法律论证，包括股权、董事会制度、上市、并购、投资等
* 风控与争议解决事务：包括积极的索赔和争议、外部和内部调查、诉讼、仲裁等
* 内部规章合规事务（人事部门作为 Owner）：企业内部各种形式的规章制度，包括[商业行为准则](../people/restrain)、保密协议

## 法则

* 企业运行仅仅依靠规章制度是远远不够的，还必须依靠文化、道德准则和价值观
* 企业法律事务的处理能力决定企业是否能够持久，法律事务的专业化、技术化决定**企业治理结构**是否完善
* 企业治理结构的完善，意味着企业管理模式的**制度化**和企业经营的合法化
* 企业管理模式的制度化主要是指公司**权利结构**的制衡性安排
* 企业法务的首要价值是为**交易合作**创造高效运作的制度，千万不能简单陷入风险防范的误区。

* 软技能
    ```
    软件业的宏观概念
    理解软件和软件源码之间的关系

    目标驱动的工作理念（非技术驱动或学习成长驱动）
    沟通中的主动权
    搜寻知识的能力
    搜寻人的能力
    协调公司资源的能力
    ```

* 硬技能

    ```

    ```
