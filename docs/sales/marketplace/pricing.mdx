# 定价

比较合适的定价模式是与 CPU 核数成一定的函数关系。目前 Azure 平台会基于 1 核 CPU，给出其他多 CPU 的建议定价。

海外和国内的定价有很大的差异，基本原因：



- 海外云支持按CPU定价，即有钱的用户多付钱
- 海外云更愿意向软件付费
- 海外云按小时付费的模式更为常见



基于差异，则所定的价格体系有很大的不同。


## 海外



国外用户习惯按小时使用云服务器，因此我们以小时单价为基础：



1核CPU基本的小时价格为：0.039美元/小时

n核CPU基本的小时价格为：(n×0.039美元/小时) 平方根



月价=小时价×24×30

年价=小时价×24×365





针对于特殊软件（品类偏，安装难，用户大）可以有针对性的定价，比如：××美元/小时



## 国内



基本的三个阶梯型的小时价：0.18元/小时，0.24元/小时，0.30元/小时

基本的三个阶梯型的包月价：15元/月，20元/月，30元/月



年价=月价×12



针对于特殊软件（品类偏，安装难，用户大）可以有针对性的定价，比如：100元/月，0.9元/小时
