# 渠道

我们的商业成功与外部伙伴密不可分，与我们的业务息息相关的外部伙伴资源包括：



### 云平台



我们当前合作的云厂商伙伴有：AWS，Azure，阿里云，华为云，腾讯云，百度云，青云



### 供应商



供应商是为我们提供软件组件或成熟软件产品的上游企业，我们通过转授或组合再转授的方式销售供应商的产品，获得客流和收益。



我们尽量针对每一个供应商，都建立良好的商务和法务合作关系，务必使得合作有据可依，合作深度和广度达到一个比较合理的状态。



我们签约的合作伙伴包括：Seafile, 明道



### 合作伙伴



我们为最终客户提供的价值可以传递到合作伙伴，让合作伙伴可以为其最终客户提供同等的价值。



当前 Websoft9 合作伙伴分为以下三种类型：



- **渠道合作伙伴**即转售 Websoft9 产品/服务；
- **技术合作伙伴**即集成 Websoft9 产品/服务，向客户提供整体方案，如 SI 或行业解决方案商；
- **独立软件合作伙伴**即通过 Websoft9 将产品发布到主流云市场的软件公司，通常称之为 ISV；

Websoft9 商业的本质是软件流通业（或者称之为软件供应链） ，流通的链条与传统的 IT 流通类似：厂家-总代-代理-客户。

我们处于努力跻身总代的位置，为价值链上的厂商和代理商提供更有价值的服务。  

IT 流通市场的典型公司有：

* 巨头：神州数码、联强 、SYNNEX、佳节、英迈
* 桌面软件分销：伊登、思杰马克丁软件、深圳市立诺软件

## 代理给谁服务？

一般来说，代理商的主要是面向政府、事业单位和大中型企业销售。 

他们的客户名单可以从：

* 招投标网站获得销售线索（政府和事业单位项目）
