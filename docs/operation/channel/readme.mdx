# 渠道

Websoft9 商业的本质是软件流通业（或者称之为软件供应链） ，流通的链条与传统的 IT 流通类似：厂家-总代-代理-客户。

我们处于努力跻身总代的位置，为价值链上的厂商和代理商提供更有价值的服务。  

IT 流通市场的典型公司有：

* 巨头：神州数码、联强 、SYNNEX、佳节、英迈
* 桌面软件分销：伊登、思杰马克丁软件、深圳市立诺软件

## 谁在做代理？

谁在做代理（面向客户的一线卖家），这些信息很难通过不会公开的，这是他们的核心资源。
那么，如何才能获取正在面向客户的一线卖家呢？这个其实非常简单，主要通过厂家的官网获取。

## 代理给谁服务？

一般来说，代理商的主要是面向政府、事业单位和大中型企业销售。 

他们的客户名单可以从：

* 招投标网站获得销售线索（政府和事业单位项目）
* 产品的官方网站，查看其最近的新闻或成功案例
