# 后勤

## 做什么？

后勤即 Back Office，它主要工作分为如下几类：


1. [7S](https://zhuanlan.zhihu.com/p/498568811) 内务管理：创造一个有序、绿色、环保、舒适的办公室环境
2. 采购执行：广泛寻源，花合适的钱买到最合适的商品或服务
3. 事务支持：为业务部门提供事务性后台支持服务 

<DocCardList items={useCurrentSidebarCategory().items}/>

## 有什么价值？

后勤事务用心钻研，也可以做的非常出色：

* 规范可靠的共享后台支持服务，每年为公司节省数万元的开支
* 创造 5S 级别的工作环境，提升企业形象，提高工作效率
* 保障业务开展，让内外协同，全局顺畅

## 配套

* 微盘**财税与后勤**空间：存放不适合公开的资料
* [Company 项目 on Github](https://github.com/websoft9op/company)：任务索引、清单和过程管理
* 供应商的信息维护在 CustomerSS 中的【运营外包商】分类中
