# 资质


## 做什么？

资质是一种无形的资产。那么，获得对企业有价值的资质以及充分利用资质带来回报是此项工作的重心。  

资质的申请需要专业公司协助，目前主要与下面的几家公司达成合作：

- [湖南大科城知识产权服务有限公司](http://www.yueluip.com/)：高企、软著、专利等
- [湖南大科城博士科技有限公司](http://www.doctortech.com.cn/)：科技市场运营、专精特效发展以及各种政府补贴项目申报等
