# 补贴

## 做什么？

中国企业的发展，离不开政府和政策的支持。很多上市公司每年获得的补贴可能超过数千万:

* [中科软关于2020年度获得政府补助的公告](http://static.sse.com.cn/disclosure/listedinfo/announcement/c/new/2021-04-16/603927_20210416_17.pdf)
* [亿嘉和科技股份有限公司关于 2022 年度获得政府补助情况的公告](http://www.sse.com.cn/disclosure/listedinfo/announcement/c/new/2023-01-10/603666_20230110_4T4C.pdf)

获得补贴除了需要熟悉政策以及申报能力之外，还需要企业具备一些硬指标：

- 税收缴纳
- 吸收就业
- 营收规模
- 企业等级（[大、中、小型企业划分参考](http://www.gov.cn/zwgk/2011-07/04/content_1898747.htm)）
- 科技属性（是否是专精特新、是否在科技园区、是否是重点产业）

为了快速的获得国家补贴或奖励政策，企业需要建立比较合适的信息渠道：

* 加入国家科技园区微信群
* 企业注册地址为科技园区
* 积极参加政策宣讲会

## 场景

### 省级-研发经费奖补{#drfee}

* 项目全称：湖南省企业研发财政奖补资金
* 概要：以上下年度研发经费的差额的10%奖补，最高1000万；
* 主管单位：湖南省科技厅.资配处
* 时间：每年 Q1 等待通知
* 门槛：无
* 详情：参考《[关于开展2021年企业研发财政奖补资金申报工作的通知](http://kjt.hunan.gov.cn/kjt/xxgk/xmxx/xmsb/202103/t20210318_14886405.html)》

### 省级-技术交易服务补助

* 项目全称：湖南省技术交易和技术转移服务补助项目
* 概要：技术交易合同收款额的1%左右
* 主管单位：湖南省科技厅.科技成果与技术市场处 
* 时间：每年11月
* 门槛：无
* 系统：[合国技术合同认定登记系统](http://210.76.97.54/index.jsp)
* 详情：参考《[关于申报2020年度湖南省技术交易和技术转移服务补助项目的通知](http://kjt.hunan.gov.cn/kjt/xxgk/tzgg/tzgg_1/202010/t20201027_13939209.html)》
* 注意事项：
    ```
    1. 只对省内（湖南省）客户提供技术咨询和技术服务的合同进行补助；
    2. 以实际收款金额为基准进行补助，累计需达到100万元，收款额与发票额相符；
    3. 必须在合同期限内完成登记备案，合同期限可为一年或几年，必须标合同总金额，但付款方式可以为分期付款，按月、季、年等；
    4. 每种技术只能认定一次，即一种技术，一个客户，一个合同；
    5. 技术合同备案对应的收入是不免税的，增值税和所得税正常缴纳。
    ```

### 市级-高企认定补贴

* 项目全称：认定高新技术企业奖补
* 概要：以上下年度研发经费的差额的10%奖补，最高1000万；
* 主管单位：湖南省科技厅.资配处
* 时间：每年 Q1 等待通知
* 系统：[长沙市科技局综合信息管理系统](http://175.6.46.174:8088/Admin/Default.aspx)、 [高企/科技型中小企业信息查询](https://fuwu.most.gov.cn/)
* 详情：参考《[关于开展2021年企业研发财政奖补资金申报工作的通知](http://kjt.hunan.gov.cn/kjt/xxgk/xmxx/xmsb/202103/t20210318_14886405.html)》

### 市级-科技计划项目

* 项目全称：长沙市科技计划项目
* 概要：针对国家重点扶持的产业（专业）进行科技研发的政策支持
* 主管单位：长沙市科学技术局.科技成果转化服务中心企业与项目服务科
* 时间：每年6月
* 门槛：无，竞争性
* 详情：参考《[关于发布2021年长沙市科技计划项目申报指南的通知 ](https://www.sohu.com/a/468835652_121106908)》

### 市级-稳岗补贴

稳岗补贴属于特殊经济形势下的特殊政策，具体以通知为准。

### 市级-人才补贴

* 项目全称：[《长沙市争创国家吸引集聚人才平台若干政策(试行)》（简称“升级版45条”）](http://www.changsha.gov.cn/szf/ztzl/qshcsydd/qshzdbd/202204/t20220429_10550135.html)
* 儿项目：[岳麓山大学科技城关于打造科技人才高地的实施办法（试行）](http://dkcte.com/wcm/dkckjcg/html/kjzc/20220726/164759939.shtml)
* 实施时间：2022年5月1日正式施行
* 主管单位：长沙市委市政府
* 无门槛：毕业2年内的本科及以上毕业生可以获得租房和生活补助（博士3万元，硕士2万元、本科1.2万元）。博士、硕士毕业生在长工作首次购房（博士6万元、硕士3万元）
