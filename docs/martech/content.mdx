# 数字营销

## 干什么？

传统的企业依靠销售人员地面部队，建立渠道网络和大客户直销，便可以在市场中占有一席之地。如今，这样的销售方法已经远远不够，因为客户更多通过网络和社交媒体去了解产品。

如何让产品驱动业务（PLG）？toB的企业营销技术（Martech）应运而生。

MarTech 是一种制作各种产品来创造营销的技术，包括广告、内容、社交、电商、数据等

Martech 在国外发展得非常成熟，与之相关的 SaaS 化软件琳琅满目，举例：

- 网站：webflow, Wix, weebly, strikingly, SQUARESPACE, shopify, duda
- 邮件营销：sendgrid, stripo, edmdesigner, postcards
- 表单采集：Typeform, Najinform, Jotform, Surveyanyplace
- 在线聊天：drift
- 电子商务：Magento, Shoplify, PrestaShop, Saleor, Bagisto



甚至还有一些将 Martech 与 企业管理系统全部打通的软件，比如：Zoho, Salesforce, WordPress 体系。

Regalinx（美国一家技术和市场调研公司）调查研究，有效的内容营销资产包括8项：

- 网站内容
- 在线会议
- 案例研究
- 电子书
- 博客内容
- 在线视频
- 社交内容
- 产品演示

数字化营销=内容营销+广泛的数字化传播渠道，因此其重点在于内容生产和内容传播，甚至内容效果的有效评估。

## 怎么做？

### 内容主数据

内容主数据极具价值，它定义了内容营销的基础数据，使得企业各种内容可以通过主数据进行**装配**后形成客户所需的内容。  

Websoft9 目前采用 Contentful 做为内容主数据的平台，其他内容渠道将基于此数据进行加工和展示。

腾讯云社区的特点调研 

文章创作：发布时可以选择可以评论和禁止评论功能。禁止评论的文章没有评论区。对外链宽容度较高。可以在每一篇文章插入商品页面链接、店铺链接等。有“关注阅读全文”的功能适合长篇知识性文章使用。

问答：提问必须是问号结尾，有字数限制，需要审核。部分问题可能不能过审。每日提问上限5条，但是一直提问可以到15条左右，能正常发布。回答需要审核，可以带商品页面外链。自己回答自己的问题一般不能成功。

评论：具有较大的随机性。多次评论相同内容会被屏蔽。需要每天评论时在页面停留五分钟左右检查评论是否能存留。
有时可以带商品页面链接，有时知识库都会被屏蔽。
