# 内容生产

## 关于

产品型企业需要通过内容去建立用户认知与培养品牌，对于面向企业的软件产品显得更为重要。  

数字化营销=内容生产+广泛的数字化传播渠道，通过内容与用户互动，最终转换为品牌认知或直接的订单。  

Regalinx（美国一家技术和市场调研公司）调查研究，软件 ISV 有效的内容营销资产包括8项：

- 内容主数据
- 网站
- 在线会议
- 案例研究
- 电子书
- 博客内容
- 在线视频
- 社交内容
- 产品演示

## 内容训练{#createcontent}

创造内容所需的核心要点：严谨的规划、简洁易读的文字、有条有理的逻辑。创作内容需要有大量的阅读，最好是逻辑性严谨的结构化知识。

### 自创训练

我们要在团队中形成良好的内容创造氛围和技巧。以创造产品指南为例，我们创造了一个**口诀**：

```
构思应在前，大纲随后到
一二三四五，事事列清楚
标题切莫长，长句要少用，
中文写完后，英文才动手
翻译过程中，图片先替换
图床应统一，路径差异小
链接细检查，错字排查掉
熟练三百日，下笔如有神
```
