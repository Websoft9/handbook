# SEO

虽然 Marketplace 是 Websoft9 重要的根据地，但私域的培养与建立是企业增长的最重要引擎。私域中最值得投入的就是 SEO。

## 要点

SEO 中的关键知识点包括：

- 搜索引擎原理：爬虫抓取、收录、链接分析、搜索结果呈现等
- 关键词研究：核心关键词、长尾关键词等
- 站内优化：网站结构、页面、标题、meta、图片表示等元素更好的适应搜索引擎
- 站外优化：通过推广、外链等提高信任度
- 监控追踪：通过数据和工具持续优化
- 工具集：各种工具的学习
